## Telescope

这是一个集成了很多可以进行搜索查询的一个插件，比如可以搜索现在的**警告信息**、**对文件进行模糊搜索**等。

### 快捷键信息

| 快捷键      | 解释                             |
|-------------|----------------------------------|
| `<space>sf` | 对文件进行模糊搜索               |
| `<space>sd` | 对警告信息进行搜索               |
| `<space>sg` | 对文件内容进行模糊搜索           |
| `<space>s.`  | 对最近打开的文件进行查找         |
| `<space>s/`  | 对现在打开的文件进行内容模糊搜索 |
| `<space>sw`  | 对现在光标所在的位置进行模糊搜索 |
